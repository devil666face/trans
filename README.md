Google Translate CLI

USAGE

COMMANDS

    completion [command]        Generate the autocompletion script for the specified shell
    help [command]              Help about any command
    list                        List available languages
    translate [text] [--flags]  Translate input from stdin or argument

FLAGS

    -h --help                   Help for trans
    -v --version                Version for trans

Show a list of all available languages that can be used as source or target for translation.

USAGE

    trans list [--flags]

EXAMPLES

    # Show all available languages
    trans list

    # The same, but using the command alias
    trans l

FLAGS

    -h --help  Help for list
